9. 🧮 Cost Calculator Logic
Formula Example:
Base = sqFt * basePricePerSqFt
RoomCost = rooms * roomMultiplier
MaterialCost = materialMultiplier * Base
LocationCost = cityMultiplier * Base

TotalEstimate = Base + RoomCost + MaterialCost + LocationCost

Logic handled in:

/services/calculatorService.js
